# 📖 Overview

ShopVerse is a modern and fully responsive e-commerce web application built with React and Redux Toolkit. It provides a complete online shopping experience, including user authentication, wishlist management, shopping cart, secure checkout, profile management, PDF receipt generation, and responsive layouts optimized for desktop, tablet, and mobile devices.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/AwaisAliMuhammad/ShopVerse.git
```

Go inside the project

```bash
cd ShopVerse
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build production version

```bash
npm run build
```
